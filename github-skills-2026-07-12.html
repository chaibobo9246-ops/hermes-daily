📋 详细说明

getfokus-app/agent-skills 是 Fokus 官方发布的 AI Agent Skill 集合， 专为 OpenClaw 和 Hermes Agent 设计。基于 @fokus-app/mcp CLI 构建， 提供 44 个工具覆盖任务管理、日历、笔记、目标追踪、项目和 AI 自动排程。

Hermes Agent 用户可通过两种方式集成：
① MCP Server（推荐）— 在 ~/.hermes/config.yaml 中配置 mcp_servers
② Skill 文件— 将 skills/fokus/ 放入 ~/.hermes/skills/

🛠 需要: Node.js 20+ 📦 安装: npm i -g @fokus-app/mcp 🔑 首次需人工登录: fokus-mcp login
